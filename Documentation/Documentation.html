Requirements

  • Code Editor: Visual Studio Code / Sublime Text
  • Browser: Chrome, Firefox, Edge, Safari (latest versions)
  • Bootstrap 5.3.2 included via CDN
  • Basic HTML & CSS knowledge

Installation

  1. Unzip the downloaded package
  2. Open the index.html file in your browser
  3. To edit, open files in your code editor
  4. Upload all files to your web server to make it live
Always keep the folder structure intact for smooth functioning.

Introduction

This template is built using HTML5, CSS3, Bootstrap 5, and JavaScript. JavaScript is included for interactive features such as sliders, navigation effects, and animations.

  • Bootstrap 5 grid system and components for seamless layouts
  • Fully responsive design optimized for all devices
  • Interactive sliders using Bootstrap Carousel or Owl Carousel
  • Cross-browser compatibility for consistent performance

Scripts & Plugins

  • Bootstrap 5 JS for navbar, modals, and carousel functionality
  • Optional Owl Carousel for advanced sliders

// Example: Initialize Bootstrap Carousel
var $comfortOwl = $(".owl-carousel").owlCarousel({
  loop: true,
  margin: 25,
  autoplay: true,
  autoplayTimeout: 5500,
  autoplayHoverPause: true,
  dots: false,
  nav: false,
  responsive: {
    0: { items: 1 },
    600: { items: 2 },
    992: { items: 3 },
    1200: { items: 4 }
  },
});

// Custom Prev/Next controls
$(".comfort-prev").click(function () {
  $comfortOwl.trigger("prev.owl.carousel");
});
$(".comfort-next").click(function () {
  $comfortOwl.trigger("next.owl.carousel");
});

Customization

  • Logo: Replace logo.png inside assets/Assets/
  • Colors: Update primary colors in assets/css/style.css
  • Fonts: Change Google Fonts link in <head>
  • Sections: Copy/paste Bootstrap grid blocks to add new content

FAQ

Q: Why are images not displaying?

A: Verify image paths in assets/Assets/ and ensure correct Bootstrap grid implementation.

Q: How do I add a new doctor profile?

A: Duplicate an existing Bootstrap card, updating the image, name, specialty, and bio.

Q: How can I add a new blog post?

A: Create a new HTML file, link it in blog.html, and use Bootstrap cards for styling.
